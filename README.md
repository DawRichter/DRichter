## Aby skompilować kod:

[Codespaces]

```
g++ [ścieżka do pliku]
```

np.

```
g++ bubblesort/bubblesort.cpp
```

po wykonaniu komendy do kompilacji zostanie utworzony plik (domyślnie a.out)

## Aby uruchomić kod:

```
./a.out
```

## Aby od razu skompilować kod i uruchomić:

```
g++ bubblesort/bubblesort.cpp && ./a.out
```
